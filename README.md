AbraFlexi Backup

![AbraFlexi Backup](abraflexi-backup.svg?raw=true)

Usage:

```shell
abraflexi-backup [company_code] [backup/directory] [path/to/.env]\n";
```

Configuration
-------------

* `ABRAFLEXI_URL`
* `ABRAFLEXI_LOGIN`
* `ABRAFLEXI_PASSWORD`
* `ABRAFLEXI_COMPANY`
* `ABRAFLEXI_BACKUP_DIRECTORY`
